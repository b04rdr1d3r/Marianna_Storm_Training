This template is designed by Blue Storm for use as a training platform for the ATAC group, expanded from the training built for the 265th TFS Hooters F-15E squadron 
ATAC comprises many squadrons (F-16, F/A-18, F-14, F-15E, Mirage 2000C) therefore this training is built with slots to accomodate all of them
The Andersen base is populated (I hate empty bases) and there is radio chatter on the frequency
There is :
- Random Air Traffic (MOOSE)
- A skynet controlled SAM site
- A supercarrier group
- A red fleet
- Multiple Tanker tracks for all systems (Boom and Basket)
- Two A/A training zone fox enabled (MOOSE) so you can call up aggressors or parry with live opponents (missiles won't kill you)
- the Ferralon bombing range is simulated at its actual location with scoring (MOOSE)
